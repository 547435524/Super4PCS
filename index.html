Super4pcs

Fast Global Pointcloud Registration via Smart Indexing

The Super4PCS library provides you with one of the most efficient way to register point-clouds acquired by any 3D acquisition device: 3D scans, Lidar, photogrammetry, etc. Clouds can be structured (range maps, 360°) or not. Normals and color are not required, but can used when available.

Super4PCS library is light, efficient, and robust to typical acquisition artifacts: missing data, noise, variation of sampling, etc ; no assumptions are made about the represented shape, and the initial poses of the clouds. And of course, all of this is open-source.

This website is currently under construction: we are preparing tutorials, test-cases, FAQ, etc. In the meantime, you may want to visit:

  • The paper project page,
  • Our github repository,
